PKBasic

my own flavor of BASIC. minimalistic, strongly-typed, and interpreted.

COMPILING

requirements: c++ standard library. tested using clang++ and g++ on OSX and ubuntu 10.04 respectively

INSTALLATION

installation is simple, just run installer.py in the root directory of the project. if you need to edit the build configuration, edit the config files in /build.
